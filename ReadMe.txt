Скрипт генерации стилей для иконок:
===================================
1.  Поисковые системы
2.  Баннерные системы
3.  Социальные сети
4.  Браузеры
5.  Операционные системы
6.  Страны
7.  Мобильные телефоны
8.  Расширения файлов
9.  Протоколы
10. Интересы

Формат иконок: только PNG 16x16 (для стран - PNG 16x11)

Для поисковыx, баннерных и социальных систем иконки скачиваются с сервиса favicon.yandex.net/favicon/
Если иконка не нашлась на сервисе, она берется из папки "if_no".

Часть иконок взята с сайтов:
http://user-agent-string.info/list-of-ua (ОС и браузеры)
http://famfamfam.com (страны - флаги)


Как добавить иконку?
====================
Добавляем новый пункт в json - ./types/xxx.json
Добавляем нужный png-файл с названием id (yyy) в папку ./images/xxx/yyy.png


Как добавить новый тип иконок?
==============================
1. Добавляем в types.json
2. Создаем файл ./types/my_type.json - список иконок
3. Создаем файл ./types/my_type.css - начальный CSS-класс для нового типа иконок
